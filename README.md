# DailyFun
A MaterialDesign Android project width MVP architecture
## 简介
一款MD风格，MVP架构的app，计划可以浏览妹子图片和知乎日报内容，目前完成妹子图片。

- 自定义ZoomImageVIew
- 使用Jsoup解析html
- 使用Glide加载图片
